# 小遥配配 B端高保真原型 - 安装说明

## 快速开始

### 1. 安装依赖

```bash
cd docs/高保真原型/B端原型/v0.1

# 使用 npm
npm install

# 或使用 pnpm（推荐）
pnpm install
```

### 2. 启动开发服务器

```bash
npm run dev
```

启动后访问：**http://localhost:5173**

## 功能预览

### 页面导航

登录后可通过侧边栏导航访问各功能模块：

| 页面 | 路由 | 功能说明 |
|------|------|----------|
| 首页 | /dashboard | 数据概览、快捷入口、最新线索 |
| SKU管理 | /sku | 电脑配置列表、搜索筛选 |
| 添加配置 | /sku/add | 动态表单添加新配置 |
| 客户线索 | /leads | 线索列表、状态筛选 |
| 线索详情 | /leads/:id | 客户详情、对话记录、跟进备注 |
| 分享管理 | /share | 二维码、链接、海报生成 |
| 个人中心 | /profile | 账号信息、修改密码 |

### 核心功能演示

#### 1. 添加电脑配置

1. 点击侧边栏「SKU管理」
2. 点击「添加配置」按钮
3. 选择设备类型（台式机/笔记本/一体机）
4. 填写配置信息（表单会根据设备类型动态变化）
5. 上传产品图片
6. 点击「保存配置」

#### 2. 查看客户线索

1. 点击侧边栏「客户线索」
2. 使用状态Tab筛选（待跟进/已联系/已成交）
3. 使用设备类型筛选
4. 搜索客户手机号或需求关键词
5. 点击线索卡片查看详情

#### 3. 跟进客户

1. 在线索详情页面查看完整对话
2. 查看「AI需求画像」了解客户需求
3. 在「跟进记录」中添加备注
4. 使用快捷按钮拨打电话或发送微信

#### 4. 生成分享海报

1. 点击侧边栏「分享管理」
2. 选择海报模板（科技风/简约风/活力风）
3. 点击「生成海报」
4. 下载海报分享到社交媒体

## 常见问题

### Q: 页面刷新后登录状态丢失？

A: 这是正常现象，原型模式下Token存储在localStorage中，可以手动修改代码实现持久化登录。

### Q: 数据为什么没有保存？

A: 当前使用Mock数据，数据仅在内存中，刷新后会重置。对接真实API后即可实现数据持久化。

### Q: 如何对接后端API？

A: 修改 `src/utils/mockData.ts` 和 `src/stores/` 中的文件，将Mock数据替换为真实的API调用。
