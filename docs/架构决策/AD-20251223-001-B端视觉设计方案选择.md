# 架构决策 - B端视觉设计方案选择

**决策日期**：2025年12月23日

## 背景

小遥配配项目需要为B端（商家后台）确定视觉设计方案。项目已完成3套不同的视觉设计原型：

1. **数据驾驶舱** - 深色科技风格，适合追求专业感和科技感的商家
2. **现代商务** - 浅色专业风格，适合追求效率和简洁的商家
3. **玻璃未来** - 新拟态玻璃风格，适合追求时尚和个性化的商家

B端用户群体主要是电脑城商家、店铺经营者，需要在日常工作中频繁使用系统进行配置管理、线索跟进等操作。系统需要提供高效、清晰、不疲劳的使用体验。

## 解决方案

选择**现代商务（Modern Business）**风格作为B端视觉设计方案。

### 设计特点
- **浅色专业风格**：采用白色/浅灰为主色调，提供清爽舒适的视觉体验
- **优雅排版**：使用Playfair Display作为标题字体，Source Sans 3作为正文字体
- **经典商务配色**：以商务蓝（#2563eb）为主色，搭配绿色、橙色、青色等辅助色
- **清晰视觉层次**：通过阴影、边框、间距建立明确的视觉层级
- **高效信息呈现**：数据卡片、列表、图表等组件设计简洁明了

### 技术实现
- 基于Vue3 + Ant Design Vue组件库
- CSS变量实现主题系统，便于后期定制
- 响应式设计，支持1280px+桌面端分辨率
- 使用系统字体栈，减少外部依赖（国内CDN fonts.loli.net）

### 组件规范
- **卡片**：16px圆角，1px边框，白色背景，hover时显示阴影
- **按钮**：主按钮使用渐变蓝色，次要按钮使用白色背景+蓝色边框
- **表单**：40px高度输入框，focus时显示蓝色边框和阴影
- **状态标签**：不同状态使用不同颜色背景和文字

## 影响

### 正面影响
1. **提升工作效率**：清晰的视觉层次减少用户认知负担，提高操作效率
2. **降低视觉疲劳**：浅色背景适合长时间使用，减少眼睛疲劳
3. **专业形象**：商务风格增强商家对系统的信任感
4. **易于维护**：基于Ant Design Vue，组件成熟稳定，后期维护成本低
5. **响应式支持**：良好的适配性支持不同屏幕尺寸

### 依赖关系
1. 需要前端团队使用Vue3 + Ant Design Vue进行开发
2. 需要设计团队输出完整的组件规范文档
3. 需要与C端视觉风格进行区分，保持品牌一致性

### 后续工作
1. 基于选定的设计方案输出完整的设计规范（颜色、字体、间距、组件等）
2. 创建Vue组件库，封装常用业务组件
3. 进行用户测试，验证设计方案的可用性
4. 完成其他页面的视觉设计（SKU管理、客户线索、分享管理、个人中心）

### 备选方案
如果后期需要支持深色模式，可以在现有浅色方案基础上添加深色主题，保持一致的设计语言。
