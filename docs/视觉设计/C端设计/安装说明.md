# 小遥配配 C端视觉设计 - 快速预览

## 快速开始

### 1. 安装依赖

```bash
cd docs/视觉设计/C端设计
npm install
```

### 2. 启动预览

```bash
npm run dev
```

浏览器将自动打开 `http://localhost:3000`

### 3. 预览设计

- 在导航页面点击任意卡片即可预览完整设计
- 点击"返回导航"按钮回到首页
