# 小遥配配 B端视觉设计 - 安装说明

## 项目结构

```
docs/视觉设计/B端设计/
├── index.html                           # 设计方案总览页
├── package.json                         # 项目配置
├── vite.config.js                       # Vite配置
├── style1-data-cockpit/                 # 方案一：数据驾驶舱（深色科技感）
│   └── DashboardPage.html
├── style2-modern-business/              # 方案二：现代商务（浅色专业感）
│   └── DashboardPage.html
└── style3-glassmorphism/                # 方案三：玻璃未来（柔和未来感）
    └── DashboardPage.html
```

## 快速预览

### 方法一：直接打开HTML文件（推荐）

直接在浏览器中打开 `index.html` 文件即可预览所有设计方案。

```bash
# Windows
start index.html

# macOS
open index.html

# Linux
xdg-open index.html
```

### 方法二：使用本地服务器

如果需要完整的开发环境体验：

```bash
# 1. 进入项目目录
cd docs/视觉设计/B端设计

# 2. 安装依赖（可选）
npm install

# 3. 启动开发服务器（需要先安装依赖）
npm run dev
```

服务器将在 `http://localhost:3001` 启动。

## 三套设计方案说明

### 方案一：数据驾驶舱（Data Cockpit）

**风格特点：**
- 深色科技风格
- 沉浸式数据可视化体验
- 霓虹色彩和发光效果
- 动态网格背景和扫描线动画
- 适合追求专业感和科技感的商家

**配色方案：**
- 主色：青色 #00f3ff
- 辅助：紫色 #a855f7、绿色 #10b981、橙色 #f59e0b
- 背景：深蓝黑色系

**字体：**
- Orbitron（标题/数字）
- Rajdhani（正文）

---

### 方案二：现代商务（Modern Business）

**风格特点：**
- 浅色专业风格
- 优雅的排版和精致的细节
- 经典商务配色
- 简洁清晰的视觉层次
- 适合追求效率和简洁的商家

**配色方案：**
- 主色：商务蓝 #2563eb
- 辅助：绿色、橙色、青色
- 背景：浅灰白色系

**字体：**
- Playfair Display（标题）
- Source Sans 3（正文）

---

### 方案三：玻璃未来（Glassmorphism）

**风格特点：**
- 新拟态玻璃风格
- 柔和的渐变背景
- 透亮质感和模糊效果
- 动态浮动光球装饰
- 适合追求时尚和个性化的商家

**配色方案：**
- 主色：粉色 #f472b6、紫色 #a855f7
- 辅助：蓝色、青色、橙色
- 背景：流动渐变色

**字体：**
- Outfit（全套）

---

## 技术栈

- **纯HTML/CSS/JavaScript** - 无需构建工具即可预览
- **CSS变量** - 便于主题定制
- **Web Fonts** - Google Fonts 字体
- **响应式设计** - 支持桌面端和移动端

## 功能模块

每个设计方案都包含B端首页（数据看板）的完整展示：

1. **侧边栏导航** - Logo、菜单、用户信息
2. **顶部导航栏** - 页面标题、日期、搜索、通知
3. **数据统计卡片** - 咨询数、线索数、转化率、成交金额
4. **快捷操作入口** - 添加配置、查看线索、分享管理
5. **最新线索列表** - 待跟进客户信息
6. **热门配置排行** - 本周热门配置统计

## 定制开发

如需基于设计方案进行Vue3 + Ant Design Vue开发：

```bash
# 1. 创建Vue3项目
npm create vite@latest xiaoyaopeipei-b-end -- --template vue

# 2. 安装依赖
cd xiaoyaopeipei-b-end
npm install

# 3. 安装Ant Design Vue
npm install ant-design-vue

# 4. 将设计方案转换为Vue组件
```
