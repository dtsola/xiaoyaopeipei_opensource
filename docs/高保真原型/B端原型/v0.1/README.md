# 小遥配配 - B端商家后台高保真原型

> AI智能导购系统 - 商家后台管理系统 v0.1

## 项目概述

小遥配配 B端商家后台是一个现代化的电脑配置管理和客户线索跟进系统。通过AI智能导购帮助商家更高效地管理产品配置和客户关系。

## 技术栈

- **框架**: Vue 3 + TypeScript
- **构建工具**: Vite
- **UI组件**: Ant Design Vue 4.x
- **状态管理**: Pinia
- **路由**: Vue Router 4
- **HTTP客户端**: Axios
- **日期处理**: Day.js

## 设计风格

采用**现代商务风格**，清爽专业：

- **主色调**: #2563eb（专业蓝）
- **字体**: Playfair Display（标题） + Source Sans 3（正文）
- **圆角**: 10px-16px
- **阴影**: 柔和的层叠阴影
- **动画**: 流畅的过渡效果

## 功能模块

### 1. 登录/注册
- Tab切换登录/注册
- 表单实时验证
- 密码强度指示器
- 记住密码功能

### 2. 首页数据看板
- 数据概览卡片（咨询数、线索数、转化率）
- 趋势对比显示
- 快捷入口
- 最新线索列表
- 热门配置排行

### 3. SKU管理
- 设备类型筛选（台式机/笔记本/一体机）
- 搜索功能
- SKU卡片展示
- 添加/编辑/删除配置
- 批量操作
- 库存管理

### 4. 添加/编辑配置（动态表单）
- 设备类型选择
- 动态表单字段（根据设备类型显示不同字段）
- 基本信息、硬件配置、价格库存、标签设置
- 图片上传
- 实时表单验证

### 5. 客户线索管理
- 状态筛选（待跟进/已联系/已成交/已放弃）
- 设备类型筛选
- 搜索功能
- 线索卡片展示
- 状态快速切换
- 新线索高亮显示

### 6. 线索详情
- 客户信息展示
- AI需求画像
- 完整对话记录
- 选中方案展示
- 跟进记录管理
- 拨打电话/发送微信快捷操作

### 7. 分享管理
- 专属二维码展示
- 二维码下载/打印
- 专属链接复制
- 海报生成（3种模板）
- 分享数据统计
- 设备类型分布图
- 预算分布图

### 8. 个人中心
- 账号信息查看
- 店铺信息编辑
- 密码修改（带强度检测）
- 使用统计
- 退出登录

## 安装运行

### 环境要求

- Node.js >= 16.x
- npm >= 8.x 或 pnpm >= 7.x

### 安装依赖

```bash
npm install
# 或
pnpm install
```

### 启动开发服务器

```bash
npm run dev
# 或
pnpm dev
```

访问 http://localhost:5173

### 构建生产版本

```bash
npm run build
# 或
pnpm build
```
