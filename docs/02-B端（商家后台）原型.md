# 小遥配配 - B端（商家后台）原型设计

根据PRD和设计标准，以下是B端原型设计（Markdown草图）：

---

## 📱 设计标准

- ✅ 符合产品定位：商家配置管理和线索跟进
- ✅ 现代化美观的UI设计：专业、高效、数据化
- ✅ 清晰的流程设计：登录 → 配置管理 → 线索跟进 → 分享推广
- ✅ 极致的用户体验：快捷操作、批量管理、数据可视化

---

## 🎨 设计风格定义

### **视觉方案：现代商务风格（Modern Business）**

采用浅色专业风格，提供清爽舒适的视觉体验，适合长时间使用。

---

### **色彩方案**

| 颜色类型 | 色值 | 应用场景 |
|---------|------|---------|
| **主色调** | #2563eb | 主按钮、链接、选中状态 |
| **背景色** | #fafbfc, #ffffff | 页面背景、卡片背景 |
| **文字色** | #1a1d23, #5a6370, #8a94a5 | 主要文字、次要文字、辅助文字 |
| **边框色** | #e8eaed, #f0f2f5 | 分割线、输入框边框 |
| **成功色** | #059669 | 成功提示、增长趋势 |
| **警告色** | #d97706 | 警告提示、待跟进状态 |
| **危险色** | #dc2626 | 错误提示、删除操作 |
| **信息色** | #0891b2 | 信息提示 |

### **字体规范**

| 字体类型 | 字体家族 | 字号 | 字重 | 应用场景 |
|---------|---------|------|------|---------|
| **标题** | Playfair Display | 28px | 600 | 页面主标题 |
| **副标题** | Playfair Display | 18px | 600 | 卡片标题 |
| **正文** | Source Sans 3 | 14px | 400/500 | 正文内容 |
| **辅助文字** | Source Sans 3 | 12px | 400 | 提示文字 |
| **按钮文字** | Source Sans 3 | 14px | 600 | 按钮标签 |
| **数据数字** | Source Sans 3 | 32px | 700 | 统计数据 |

### **组件规范**

#### **按钮**
- 主按钮：高度40px，圆角10px，渐变背景 `linear-gradient(135deg, #2563eb, #1d4ed8)`
- 次要按钮：高度40px，圆角10px，白色背景，蓝色边框

#### **卡片**
- 圆角：16px
- 边框：1px solid #e8eaed
- 阴影：0 4px 12px rgba(0, 0, 0, 0.06)
- Hover：阴影加深，轻微上浮

#### **输入框**
- 高度：40px
- 圆角：10px
- 边框：1px solid #e8eaed
- Focus：蓝色边框 + 阴影

#### **标签**
- 圆角：20px
- Padding：4px 12px
- 待跟进：橙色背景 #fef3c7，橙色文字
- 已联系：蓝色背景 #dbeafe，蓝色文字
- 已成交：绿色背景 #d1fae5，绿色文字

### **布局规范**

| 布局元素 | 尺寸 |
|---------|------|
| 侧边栏宽度 | 260px |
| 顶部导航栏高度 | 72px |
| 内容区域内边距 | 32px |
| 卡片间距 | 20px |
| 数据卡片高度 | 自适应，约120px |

1. **数据卡片动画**：数字滚动效果（从0到目标值）

3. **快捷入口hover**：图标放大、背景色变化
4. **最新线索**：每30秒自动刷新，新线索高亮3秒
5. **热门配置**：点击跳转到SKU详情

### **SKU管理页面**
1. **设备类型Tab**：切换时带滑动动画
2. **搜索框**：输入延迟300ms后触发搜索
3. **筛选下拉**：多选，支持价格区间、标签、库存状态
4. **卡片hover**：轻微上浮（translateY: -4px）、阴影加深

8. **分页**：每页显示10条，支持跳页

1. **设备类型选择**：单选，选择后动态显示对应表单

   - 限制：jpg/png，单张<2MB，最多5张

7. **保存成功**：显示成功提示，2秒后返回SKU管理页面
