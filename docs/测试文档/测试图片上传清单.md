## OSS上传命令参考

### 使用阿里云OSS控制台上传

1. 登录阿里云OSS控制台
2. 选择Bucket：`xiaoyaos-file`
3. 进入 `test/` 目录
4. 直接上传11张图片文件

### 使用OSS命令行工具（ossutil）

```bash
# 进入test目录
ossutil cp y7000p_1.jpg oss://xiaoyaos-file/test/
ossutil cp y7000p_2.jpg oss://xiaoyaos-file/test/
```
