## 2. 接口输入数据准备

### 2.1 C端接口测试数据

#### 2.1.1 对话模块 - 发送消息

**测试场景1：首次对话（空session_id）**

```json
{
  "session_id": "",
  "shop_id": "1000000000000000001",
  "message": "我想要一台6000左右的游戏笔记本能玩原神"
}
```

**测试场景2：继续对话**

```json
{
  "session_id": "session_test_001",
  "shop_id": "1000000000000000001",
  "message": "经常携带"
}
```

**测试场景3：完整信息一次性提供**

```json
{
  "session_id": "",
  "shop_id": "1000000000000000001",
  "message": "我想买一台8000左右的台式机，主要用来剪辑视频，品牌偏好华硕，需要32G内存"
}
```

**边界测试数据**

| 场景 | message值 | 预期结果 |
|------|-----------|---------|
| 空消息 | `""` | 400错误 |
| 超长消息 | 500个字符的字符串 | 正常处理 |
| 特殊字符 | `"测试<>\"'&%$#@"` | 正常处理 |
| 纯数字 | `"123456"` | 正常处理 |

#### 2.1.2 对话模块 - 获取推荐方案

**测试数据**

```json
{
  "session_id": "session_test_001",
  "shop_id": "1000000000000000001",
  "needs": {
    "budget": "6000左右",
    "device_type": "笔记本",
    "usage": "游戏",
    "requirements": "玩原神",
    "portable": "主要在家用"
  }
}
```

**needs字段组合测试**

| 组合 | budget | device_type | usage | 预期结果 |
|------|--------|-------------|-------|---------|
| 基础 | `"5000"` | `"笔记本"` | `"办公"` | 返回办公笔记本 |
| 高预算 | `"15000"` | `"台式机"` | `"游戏"` | 返回高端游戏台式机 |
| 模糊预算 | `"5000-8000"` | `"笔记本"` | `"设计"` | 返回设计本 |
| 品牌偏好 | `"10000"` | `"笔记本"` | `"游戏"` + `"联想"` | 返回联想游戏本 |

### 2.2 B端接口测试数据

#### 2.2.1 认证模块 - 商家注册

**测试数据**

```json
{
  "username": "test_shop_001",
  "password": "Test123456!",
  "shop_name": "测试电脑城001",
  "phone": "13800138001",
  "address": "北京市朝阳区测试路123号",
```

**username测试用例**

| 用户名 | 预期结果 |
|--------|---------|
| `test` | 正常（3-20字符） |
| `a` | 400错误（太短） |
| `a@b$c%d^e&f*g` | 正常（特殊符号） |
| `test_shop_001_test` | 正常（18字符） |

| 密码 | 预期结果 |
|------|---------|
| `Password123!` | 正常 |
| `pass` | 400错误（太短） |
| `password` | 400错误（无大写+数字） |
| `PASSWORD123!` | 400错误（无小写） |

```python
        merchants_data = [generator.generate_merchant(i) for i in range(1, 6)]
        merchants = [Merchant(**data) for data in merchants_data]
        db.add_all(merchants)
        db.commit()
        print(f"已插入 {len(merchants)} 个商家")

        # 2. 插入SKU
        print("正在插入SKU数据...")
        skus = []
        for merchant in merchants:
            for i in range(1, 11):  # 每个商家10个SKU
                sku_data = generator.generate_sku(merchant.id, i)
                skus.append(Sku(**sku_data))
        db.add_all(skus)
        db.commit()
        print(f"已插入 {len(skus)} 个SKU")

        # 3. 插入对话
        print("正在插入对话数据...")
        conversations = []
        for merchant in merchants:
            for i in range(1, 21):  # 每个商家20个对话
                conv_data = generator.generate_conversation(merchant.id, i)
                conversations.append(Conversation(**conv_data))
        db.add_all(conversations)
        db.commit()
        print(f"已插入 {len(conversations)} 个对话")

        # 4. 插入消息
        print("正在插入消息数据...")
        messages = []
        for conv in conversations[:10]:  # 为前10个对话生成消息
            # AI欢迎语
            messages.append(Message(**generator.generate_message(
                conv.id, "assistant", "您好！我是小遥，我来帮您选一台合适的电脑～"
            )))
            # 用户消息
            messages.append(Message(**generator.generate_message(
                conv.id, "user", "我想要一台6000左右的笔记本"
            )))
        db.add_all(messages)
        db.commit()
        print(f"已插入 {len(messages)} 条消息")

        # 5. 插入线索
        print("正在插入线索数据...")
        leads = []
        for conv in conversations[:5]:
            lead_data = generator.generate_lead(
                conv.merchant_id,
                conv.id,
                skus[0].id if skus else None
            )
            leads.append(Lead(**lead_data))
        db.add_all(leads)
        db.commit()
        print(f"已插入 {len(leads)} 条线索")

        # 6. 插入分享统计
        print("正在插入分享统计数据...")
        stats = []
        for merchant in merchants:
            stats_data = generator.generate_share_stats(merchant.id, days_back=7)
            stats.extend([ShareStat(**data) for data in stats_data])
        db.add_all(stats)
        db.commit()
        print(f"已插入 {len(stats)} 条统计数据")

        print("\n✅ 测试数据插入完成！")

    except Exception as e:
        db.rollback()
        print(f"❌ 数据插入失败: {e}")
        raise
    finally:
        db.close()

if __name__ == "__main__":
    insert_test_data()
```

```python
"""
验证测试数据是否正确插入
"""

from app.db.database import SessionLocal
from app.models import Merchant, Sku, Conversation, Message, Lead, ShareStat, AiLog

    db: Session = SessionLocal()

    try:
        print("\n==========================================")
        print("测试数据验证报告")
        print("==========================================\n")

        # 统计各表数据量

        conversation_count = db.query(Conversation).count()
        message_count = db.query(Message).count()
        lead_count = db.query(Lead).count()
```
